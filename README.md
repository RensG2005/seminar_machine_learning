## Abstract

We study whether semantic drift in 10-K filings predicts future stock returns among S&P 500 firms between 2005 and 2024. Using dense sentence embeddings generated by the all-MiniLM-L6-v2 transformer model, we compute a Semantic Drift Score for each firm-year across three sections of the 10-K: Item 1 (Business), Item 1A (Risk Factors), and Item 7 (MD&A). A long-short portfolio sorted on Item 1A drift earns a Fama-French five-factor alpha of -36.8 basis points per month (t = -2.92).

## Repository structure

- `SEC_10K_Extractor_Drive.ipynb` — downloads 10-K filings from EDGAR
- `text_preprocessing.ipynb` — parses Items 1, 1A, 7 via regex
- `semantic_drift_minilm(1).ipynb` — computes drift scores using sentence embeddings
- `pull_crsp_returns.ipynb` — pulls CRSP returns and Compustat accounting data
- `return_predictability_v3_updated.ipynb` — portfolio sorts, factor models, Fama-MacBeth
- `descriptive_statistics.ipynb` — summary tables
- `lazy_prices_dataset.csv` — final merged panel

## Data

- 10-K filings: SEC EDGAR
- Returns & accounting: CRSP and Compustat (via WRDS)
- Factors: Kenneth French data library
- Loughran-McDonald Master Dictionary (1993-2024)
